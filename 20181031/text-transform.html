Abstract

This module provides a grammar for the text-transform property in braille CSS [BRAILLECSS].

Status of This Document

This document is merely a public working draft of a potential specification. It has no official standing of any kind and does not represent the support or consensus of any standards organisation.

This document attempts to establish a grammar of keywords that can be passed as context to a braille translator to influence the translation in a predictable way. The idea is to be able to use the grammar across several standards and tools: CSS, OBFL [OBFL], Liblouis, etc. In the case of CSS the grammar applies to the text-transform property.

The document is incomplete, but should make the intend clear.
